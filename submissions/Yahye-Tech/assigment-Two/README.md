# Medical Wholesale Product Demand Dataset

## Overview

This repository contains a self-collected dataset and research paper prepared for the **Data Foundations for Machine Learning** practical assignment.

The project focuses on analyzing a medical wholesale product demand dataset and applying fundamental Machine Learning concepts such as features, labels, dataset structure, data quality assessment, and learning type.

---

## Dataset Information

* **Dataset Name:** Medical Wholesale Product Demand Dataset
* **Number of Samples:** 50
* **Number of Columns:** 9
* **Data Source:** Self-collected and organized in Microsoft Excel
* **Domain:** Medical Wholesale / Pharmaceutical Products

### Dataset Columns

1. No
2. Inventory
3. Amount
4. Avg Price
5. COGS
6. Avg COGS
7. Gross Margin
8. Gross Margin %
9. Qty Sold

---

## Machine Learning Perspective

### Features (X)

* Inventory
* Avg Price
* COGS
* Avg COGS
* Gross Margin
* Gross Margin %

### Label (y)

* Qty Sold

### Learning Type

* Supervised Learning

### Machine Learning Task

* Regression

The objective is to predict the quantity of products sold based on pricing, costs, and profit-related features.

---

## Data Quality Issues

The dataset is intentionally kept in its raw form for educational purposes. The following issues were identified:

* Typographical errors in product names
* Inconsistent text formatting
* Mixed information in a single column
* Irregular spacing
* Outliers
* Categorical information embedded within text

These issues will be addressed during future preprocessing and data cleaning stages.
